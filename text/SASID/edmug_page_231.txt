About Decisions
A decision enables you to combine rule sets, analytical models, treatment groups, 
code files, record contacts nodes, and conditional logic into a single process. You 
can also add a decision to another decision.
TIP Your site might have additional custom node types that you can add to 
a decision. You can create custom node types by using the Decision 
Management REST API. For more information, see https://
