Syntax
DATAGRID_SET  (dataGrid , column , rowNumber , variableOrValue )
Required Arguments
dataGrid
specifies the name of the data grid. This argument must be a variable of type 
DATAGRID.
column
specifies the name of the column whose value is to be set. You can specify a 
column name in single quotation marks or a variable that evaluates to a column 
name.
rowNumber
specifies the number of the row to be set. You can specify a number or a 
variable that evaluates to a number.
variableOrValue
specifies the value to be assigned to column . You can specify a number, a 
character value enclosed in single quotation marks, the name of a variable, or an 
expression. The value that you specify must be or must evaluate to the same 
data type as column .
